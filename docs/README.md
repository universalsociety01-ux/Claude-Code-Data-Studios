# Project Documentation

## Directories

| Directory | Purpose |
|-----------|---------|
| architecture/ | System architecture documentation, diagrams |
| adr/ | Architecture Decision Records |
| runbooks/ | Operational runbooks for common procedures |

## Getting Started

- See .claude/docs/quick-start.md for setup
- See .claude/docs/agent-roster.md for agent reference
- See .claude/docs/templates/ for document templates

## Adding Documentation

1. Architecture docs go in architecture/
2. Decision records go in adr/ (use template: .claude/docs/templates/adr.md)
3. Runbooks go in runbooks/ (use template: .claude/docs/templates/runbook.md)
