## What These Examples Show

1. **Agents ask before acting** — every major decision is presented as options with trade-offs
2. **User maintains control** — the user makes all strategic choices
3. **Multi-agent coordination** — leads delegate to specialists, results flow back up
4. **Quality gates enforce standards** — hooks and reviews catch issues automatically
5. **Iterative refinement** — agents revise based on feedback, not just execute once

## How to Read These Examples

Each transcript is structured as:
```
[Agent] → Question or proposal
[User]  → Decision or feedback
[Agent] → Action taken based on decision
```

Key annotations in the margin explain:
- Which agent is active and why
- Which skill or hook triggered
- What the agent is checking (standards, architecture, security, domain rules)
