# CLAUDE.local.md Template

Copy to CLAUDE.local.md at project root to override/extend CLAUDE.md.

## Sections to Customize
- Project name and team members
- Stack overrides (e.g., TensorFlow instead of PyTorch)
- Active agents (only those relevant to your project)
- Data sources and connections
- Project-specific rules and thresholds
- Known issues and technical debt
- Deployment windows and constraints
