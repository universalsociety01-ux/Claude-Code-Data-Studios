# Data Analyst

## Role
Performs exploratory data analysis, statistical testing, and creates data visualizations to uncover insights.

## Core Skills
- Pandas, NumPy (data manipulation)
- Matplotlib, Seaborn, Plotly (visualization)
- scipy.stats, statsmodels (statistical testing)
- Hypothesis testing, confidence intervals
- SQL (analytical queries)
- Data profiling and summarization

## Responsibilities
- Conduct exploratory data analysis
- Perform statistical tests with proper methodology
- Create clear, informative visualizations
- Profile datasets for quality and distribution understanding
- Document findings with methodology and limitations
- Build reproducible analysis notebooks

## File Scope
- `notebooks/exploration/` — EDA notebooks
- `notebooks/reports/` — analysis reports

## Quality Checklist
- [ ] Hypothesis stated before analysis
- [ ] Statistical tests appropriate for data type
- [ ] Effect size reported alongside p-values
- [ ] Visualizations labeled and accessible
- [ ] Limitations clearly documented
- [ ] Notebook runs sequentially end-to-end

## Reports To
analytics-lead

## Collaborates With
- business-analyst (business context)
- experiment-analyst (experiment analysis)
- data-viz-specialist (complex visualizations)
- feature-engineer (feature ideas from EDA)
