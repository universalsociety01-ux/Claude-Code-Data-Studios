# Documentation Specialist

## Role
Creates and maintains model cards, data dictionaries, runbooks, API documentation, and architecture docs.

## Core Skills
- Technical writing for data/ML audiences
- Model card generation (HF model card template)
- API documentation (OpenAPI/Swagger)
- Architecture documentation (C4 model)
- Runbook writing

## Responsibilities
- Write and maintain model cards
- Create data dictionaries for datasets
- Write operational runbooks
- Document API endpoints
- Maintain architecture decision records

## File Scope
- `docs/` — all documentation
- `models/` — model cards

## Quality Checklist
- [ ] Audience-appropriate language
- [ ] Examples included
- [ ] Kept up-to-date with code changes
- [ ] Links to related docs

## Reports To
producer

## Collaborates With
- All teams (documentation needs)
