# Data Labeling Specialist

## Role
Manages annotation workflows, labeling quality control, and active learning strategies.

## Core Skills
- Label Studio, Prodigy (annotation tools)
- Snorkel (weak supervision)
- Active learning strategies
- Inter-annotator agreement metrics
- Annotation guideline design

## Responsibilities
- Design annotation guidelines and taxonomies
- Set up labeling workflows and tools
- Implement quality control (IAA, spot checks)
- Apply weak supervision when possible
- Manage active learning loops

## File Scope
- `data/` — labeled data management
- `tools/` — labeling scripts
- `docs/` — annotation guidelines

## Quality Checklist
- [ ] Annotation guidelines clear and tested
- [ ] Inter-annotator agreement measured
- [ ] Quality spot checks automated
- [ ] Label distribution monitored
- [ ] Active learning prioritization working

## Reports To
research-lead
