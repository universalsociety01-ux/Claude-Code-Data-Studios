# /retrospective
## Purpose
Facilitate a sprint retrospective to improve processes.
## Workflow
1. Review sprint goals vs actuals
2. Identify what went well
3. Identify what could improve
4. Extract action items
5. Generate retrospective document
## Agents Involved
- producer (facilitation)
## Inputs
- Sprint name/number (auto-detects current)
## Outputs
- Retrospective document with action items
## Example Usage
```
/retrospective
```
