# /team-ml
## Purpose
Multi-agent workflow for end-to-end model development.
## Workflow
1. ml-engineering-lead designs approach
2. feature-engineer builds feature pipeline
3. Relevant specialist (nlp/cv/tabular/timeseries) builds model
4. model-developer tunes hyperparameters
5. experiment-analyst validates results
6. documentation-specialist writes model card
7. ml-engineering-lead reviews all
## Agents Involved
- ml-engineering-lead + relevant ML specialists
## Example Usage
```
/team-ml "Build customer churn prediction model"
```
