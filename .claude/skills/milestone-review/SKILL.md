# /milestone-review
## Purpose
Review progress toward current milestone across all teams.
## Workflow
1. Gather status from all workstreams
2. Assess progress against milestones
3. Identify risks and blockers
4. Update milestone tracking
5. Generate status report
## Agents Involved
- producer (coordination), all leads (status)
## Inputs
- Milestone name (auto-detects current)
## Outputs
- Milestone status report
## Example Usage
```
/milestone-review
```
