# /team-deploy
## Purpose
Multi-agent workflow for model deployment to production.
## Workflow
1. mlops-lead plans deployment
2. ml-deploy-engineer packages and containerizes
3. infrastructure-engineer provisions resources
4. monitoring-engineer sets up observability
5. ml-pipeline-engineer configures retraining
6. testing-specialist runs integration tests
7. mlops-lead approves go-live
## Agents Involved
- mlops-lead + all MLOps specialists
## Example Usage
```
/team-deploy "Deploy fraud detection model v3 to production"
```
