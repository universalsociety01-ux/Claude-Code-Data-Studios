# /team-analysis
## Purpose
Multi-agent workflow for comprehensive data analysis.
## Workflow
1. analytics-lead scopes analysis
2. data-analyst performs EDA
3. experiment-analyst designs tests (if needed)
4. product-analyst adds product context
5. business-analyst creates stakeholder report
6. data-viz-specialist polishes visualizations
## Agents Involved
- analytics-lead + all analytics specialists
## Example Usage
```
/team-analysis "Investigate Q4 revenue drop in APAC region"
```
